# Air Quality Analysis Dashboard ☁️

## Deskripsi
Proyek ini merupakan analisis data kualitas udara yang mencakup tren polusi PM2.5 dan korelasi antara faktor cuaca dengan kadar polutan. Proyek ini dikembangkan sebagai bagian dari submission kelas "Belajar Analisis Data dengan Python" di Dicoding.

## Struktur Proyek
- `dashboard/`: Berisi kode dashboard Streamlit dan dataset yang telah dibersihkan.
- `data/`: Berisi dataset mentah (raw data).
- `notebook.ipynb`: Dokumentasi lengkap proses analisis data (Gathering hingga Conclusion).
- `requirements.txt`: Daftar library yang dibutuhkan untuk menjalankan proyek.
- `url.txt`: Link dashboard yang telah di-deploy ke Streamlit Cloud.
